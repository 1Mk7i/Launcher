1. Створити новий проєкт mysite-react
2. Переконатись, що встановлено останні версії:
- Node.js (команда node -v) v20.8.0 і вище
- npm (команда npm -v) v10.1.0 і вище
- git (команда git --version)
- github CLI (команда gh --version)
3. Створити react app за допомогою команди npx create-react-app .
4. Встановити пакети за допомогою команди npm install
5. Переконатись, що додаток працює за допомогою команди npm start
6. Переконатись, що працює hot reload. Для цього змінити функцію App в файлі App.js і переконатись, що зміни відображаються в браузері. Ви можете змінити щось в HTML, який генерує ця функція.
7. Створимо локальний репозиторій за допомогою команди git init
8. Додамо всі файли в локальний репозиторій за допомогою команди git add .
9. Зробимо перший коміт за допомогою команди git commit -m "Initial commit"
10. Авторизуємось на github.com в браузері
11. Авторизуємося в github за допомогою команди gh auth login:
- Виберіть GitHub.com
- Виберіть HTTPS
- На питання 'Authenticate Git with your GitHub credentials?' відповідь Y
- Копіюємо код з консолі, натискаємо Enter; Відкриється браузер, де потрібно вставити код і натиснути Enter і дозволити авторизацію.
12. Створюємо новий репозиторій на github за допомогою команди gh repo create "Назва репозиторію" --public
13. Додаємо віддалений репозиторій за допомогою команди git remote add origin "Посилання на репозиторій"
Приклад: git remote add origin https://github.com/SurkovKostiantyn/kn22react.git
14. Пушимо наш локальний репозиторій на github за допомогою команди git push origin master
15. Перевіряємо, що наш репозиторій з'явився на github
16. Інсталюємо gh-pages за допомогою команди npm install gh-pages --save-dev
17. Відкриваємо файл package.json і під записом "name": "mysite-react" додаємо запис
"homepage": "https://ваш_логін_на_гітхаб.github.io/ваш_репозиторій"
Приклад:
"homepage": "https://SurkovKostiantyn.github.io/mysite-react"
Далі додаємо записи в scripts в цьому ж файлі:
"predeploy": "npm run build",
"deploy": "gh-pages -d build"
18. Прописуємо команду npm run deploy
18. Через 2-3 хвилини заходимо на сайті github в ваш репозиторій, Settings -> Pages -> Branch -> gh-pages branch - тобто обираємо гілку, з якої буде відображатись наш сайт.
19. Чекаємо 2-3 хвилини перевіряємо, що сайт відображається за адресою https://ваш_логін_на_гітхаб.github.io/ваш_репозиторій
